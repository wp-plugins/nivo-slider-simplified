Nivo Slider Simplified


== Description ==

Nivo Slider Simplified is slider using jquery , you can use drag drop you slide to slider , can use multi instance and setting to set the default features.
This plugin uses's jquery of nivo slider thanks to Nivo Slider Team.

<h3>To add Slider to your website </h3>
<h3> Method 1</h3>use short code <code>[nivo_slider_simplified id='X']</code> to your page or post or text widget ,X is slider id
<h3> Method 3</h3>to use in theme use <code>&lt;?php echo do_shortcode('[nivo_slider_simplified id='X']'); ?&gt;</code> to your template 
	


== Installation ==

The plugin can be use just by installing it by wordpress admin or upload manually. 

1. Upload `nivo-slider-simplified` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now go to nivo-slider-simplified tab at wp-admin
